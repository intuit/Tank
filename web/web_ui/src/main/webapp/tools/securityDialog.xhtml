Since Java 7 update 51, Java does not allow applets to run with self signed certificates. If you see this screen or something like it:
You will need to import the certificates used to sign the applets if you want to run as an applet. You can still run the tools by downloading them and running locally.
  1. Download this certificate file: self_signed_tank.p12
  2. Open the Java Control Panel.
    • Mac - From System Preferences, Other section, click on Java
    • Windows - From Control Panels, click on Java
  3. Go To Security tab and click on Managed Certificates.
  4. Select the Certificate type as Signer CA.
  5. Click Import and select the self_signed_tank.p12 file you downloaded earlier.
  6. Restart your browser and launch the applet again.
NOTE** If you are running Safari on Mac, you will need to set the plug in to run java in unsafe mode for this site.
  1. Open Safari Preferences and go to Security Tab
  2. Click on on Manage Website Settings
  3. Click on the java plugin and set the current open website to run in unsafe mode.
